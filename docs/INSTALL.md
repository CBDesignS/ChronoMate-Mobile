# ChronoMate Android Installation Guide

## Introduction

Thank you for downloading **ChronoMate Android**.

ChronoMate Android is a lightweight, offline chronograph recording application designed specifically for air rifle shooters.

The application installs as a normal Android app (APK) and stores all of your data locally on your own device.

No account, subscription or internet connection is required.

---

## System Requirements

ChronoMate Android requires:

- Android 5.1 or later
- Amazon Fire OS supported
- Approximately 25 MB of free storage

The application has been tested on:

- Amazon Fire HD 10 (7th Generation)
- Amazon Fire HD 10 (11th Generation)
- Google Pixel 10 (Android 17)

---

## Installing ChronoMate

1. Download the latest APK from the GitHub Releases page.

2. If prompted, allow installation from unknown sources.

3. Open the downloaded APK.

4. Select **Install**.

5. Once installation has completed, open **ChronoMate** from your app drawer.

The application is now ready to use.

---

## Amazon Fire Tablets

Amazon Fire tablets may display a warning before installing applications that were not downloaded from the Amazon Appstore.

This is normal.

Simply confirm that you wish to continue with the installation.

---

## Google Play Protect

Because ChronoMate is distributed directly from GitHub rather than the Google Play Store, Android may prevent the application from being installed while **Google Play Protect** is enabled.

If this happens:

1. Open the **Google Play Store**.
2. Tap your **account/profile icon**.
3. Select **Play Protect**.
4. Tap the **Settings** (cog) icon in the top-right corner.
5. Turn **Scan apps with Play Protect** **Off**.
6. When prompted, choose **Pause**.

Choosing **Pause** automatically re-enables Play Protect after approximately 24 hours and is the recommended option.

Once Play Protect has been paused:

1. Return to the downloaded ChronoMate APK.
2. Install the application.
3. After installation, Play Protect will automatically be enabled again after the 24-hour pause expires.

> **Note:** You can disable Play Protect permanently if you wish, but using the temporary **Pause** option is recommended.
>

## Updating ChronoMate

Updating is straightforward.

1. Export a backup (recommended).
2. Download the latest APK.
3. Install the new APK over the existing installation.
If Android prevents the update, refer to the **Google Play Protect** section above.

Your rifles, pellets and settings will normally be preserved.

Creating a backup before every update is still recommended.

---

## Uninstalling

ChronoMate can be removed like any normal Android application.

If you wish to keep your data before uninstalling:

1. Open ChronoMate.
2. Select **Export Data**.
3. Save the backup file somewhere safe.

After reinstalling, simply use **Import Data** to restore everything.

---

## Printing

ChronoMate uses Android's built-in **Print Manager**.

Depending on your device, you can:

- Print directly to a printer.
- Save reports as PDF.
- Share reports using Android's print services.

---

## Troubleshooting

### Installation blocked

Allow installation from unknown sources or follow the on-screen prompts.

### Play Protect prevents installation

Temporarily pause **Google Play Protect** as described in the **Google Play Protect** section above, then install the APK.

### Updating fails

Ensure you are installing a newer version over an existing ChronoMate installation.

### Import problems during testing

If Android Developer Options are enabled, ensure **Don't keep activities** is **disabled**.

Enabling this option forces Android to close applications whenever they lose focus and can interfere with the Android file picker.
